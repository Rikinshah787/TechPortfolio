# TechPortfolio

A creative 3D portfolio website for **Rikin Shah** — Project Manager, Engineer, and Creative Builder.

## Features

- **Custom 3D Cartoon Avatar** — Programmatic Three.js character with cel-shaded toon materials, idle animations, blink system, and cursor-tracking eyes
- **GSAP Scroll Animations** — Smooth scroll-triggered transitions with ScrollSmoother and SplitText
- **Physics-Based Tech Stack** — Interactive 3D spheres with Rapier physics engine
- **Dark + Emerald/Teal Aesthetic** — Modern color scheme with glowing accents
- **Fully Responsive** — Optimized for desktop, tablet, and mobile

## Tech Stack

React, TypeScript, Three.js, GSAP, React Three Fiber, Rapier Physics, Vite

## Getting Started

```bash
npm install
npm run dev
```
